# Autonomous Warehouse Surveillance

ROS-based autonomous surveillance system for warehouse environments.

## Structure

- **[offboard/ros_workspace](offboard/ros_workspace)** - Offboard control and monitoring system
- **[onboard/ros_workspace](onboard/ros_workspace)** - Onboard drone/robot control system

## Setup

Navigate to the respective workspace directories for setup and usage instructions.

```bash
# Offboard setup
cd offboard/ros_workspace

# Onboard setup
cd onboard/ros_workspace
```
